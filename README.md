# SE Project

### 1. Clone repo

### 2. Create .env file
- Create .env file in project folder
- Enter these lines to that:

```
MONGODB_URL=mongodb URL
JWT_SECRET = secretkey

```

### 3. Run Backend

```
$ npm install
$ npm run build
$ npm start
```

### 4. Run Frontend

```
# open new terminal
$ cd frontend
$ npm install
$ npm start
```
